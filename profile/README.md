### Admin Rules
#### Checklist For Main Branch Protection:
- [ ] Require pull request before merge
- [ ] Require min. 1 approval
- [ ] Dismiss stale pull request approvals
- [ ] Require conversation resolution
- [ ] Lock branch
- [ ] Do not allow bypassing
